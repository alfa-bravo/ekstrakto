ekstrakto
=====
extract dominant colors from images

installation
-----
```sh
pip install ekstrakto
```

usage
-----

_command line utility_
```sh
ek --help
...
ek <options> image
```
_e.g. analyze an image file and display the colors_
```sh
ek.py --display image
```
_for developers_
```sh
cd path/to/ekstrakto
py -3 ./bin/ek.py <options> image
```
